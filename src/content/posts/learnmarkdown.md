> 记录一下一些markdown基本的语法

---

## 1. 标题

```md
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

---

## 2. 段落与换行

- 段落：直接空一行就是新段落
- 换行：多数 Markdown 里「行尾两个空格 + 回车」可强制换行

```md
第一段。

第二段（与上一段中间空一行）。

这一行末尾加两个空格  
就会换行（取决于渲染器设置）。
```

---

## 3. 强调：粗体/斜体/删除线

```md
*斜体* 或 _斜体_
**粗体** 或 __粗体__
***粗斜体***
~~删除线~~
```

---

## 4. 引用

```md
> 这是一段引用
>
> 引用也可以分段
>> 也支持嵌套引用
```

---

## 5. 列表

### 5.1 无序列表

```md
- 项目一
- 项目二
  - 子项目（缩进 2~4 个空格）
```

### 5.2 有序列表

```md
1. 第一步
2. 第二步
3. 第三步
```

### 5.3 任务列表

```md
- [ ] 未完成
- [x] 已完成
```

---

## 6. 代码：行内代码与代码块

### 6.1 行内代码

```md
使用 `pnpm dev` 启动开发服务器。
```

### 6.2 代码块

> 常见语言：`js` `ts` `bash` `json` `html` `css` `md` `python`

````md
```js
function sum(a, b) {
  return a + b;
}
```
````

---

## 7. 链接

```md
[文字链接](https://example.com)
[带标题的链接](https://example.com "提示文本")
```

### 7.1 引用式链接

```md
这是一个链接：[我的站点][site]

[site]: https://example.com
```

---

## 8. 图片

```md
![图片说明](https://example.com/image.png)
```

> 本地图片是否可用取决于博客框架/构建配置（有的需要放到指定目录）。

---

## 9. 分割线

```md
---
```

---

## 10. 表格

```md
| 列1 | 列2 | 列3 |
| --- | --- | --- |
| a   | b   | c   |
| 1   | 2   | 3   |
```

对齐：

```md
| 左对齐 | 居中 | 右对齐 |
| :----- | :--: | -----: |
| a      | b    | c      |
```

---

## 11. 转义

当你想显示 Markdown 符号本身时，用反斜杠：

```md
\* 这不是斜体 \*
\# 这不是标题
```

---

## 12. html

很多渲染器允许混用少量 HTML：

```html
<p>这是一段 HTML 段落</p>
<br />
```

> 是否支持取决于站点的 Markdown 渲染配置（有的会禁用部分 HTML）。

---

## 13. 最小写作模板

````md
---
title: 文章标题
published: xxxx-xx-xx
description: 一句话介绍文章内容
tags: [tag1, tag2]
category: notes
draft: false
---

> 小引用写明写作目的

## 小节标题 1
正文...

## 小节标题 2
- 要点 1
- 要点 2

```bash
# 示例命令
echo "hello,world"
rm -rf /* (bushi)
```
````
